## alpaca-trader

A CLI tool for the Alpaca Trading API

### Synopsis

alpaca-trader is a powerful command-line interface for interacting
with the Alpaca Trading API to manage your retail/paper account, positions, and orders.

Example:
  alpaca-trader account get --env paper
  alpaca-trader positions list

Set APCA_API_KEY_ID and APCA_API_SECRET_KEY in your environment to authenticate.

### Options

```
      --all                 Automatically fetch all pages for list endpoints
      --api-key string      Alpaca API Key ID
      --api-secret string   Alpaca API Secret Key
      --config string       config file (default is $HOME/.alpaca-trader.yaml)
      --env string          Alpaca environment (paper or live) (default "paper")
  -h, --help                help for alpaca-trader
      --output string       Output format (table or json) (default "table")
      --query string        Filter output using jq-like syntax (forces json output if used)
```
